## 簡介

重機是本人感興趣的主題，正好作為 Vue3 前端技術練習，其主要功能如下：

- 前台:
  1. 最新消息
  2. 租賃車種總覽 (搜尋車種、單一車款介紹等)
  3. 重機販售
  4. 預約租車/賞車
  5. 加入收藏
- 後台: 頁面製作當中

## 使用技術

- 設計:

  - Figma => [設計稿連結](https://www.figma.com/file/ybOB9cYwU7M4g7waa290lZ/RentRide?node-id=0-1&t=XF3iV9zNB0v8TPb6-0)

  - Vue3
  - VueRouter
  - Pinia
  - JavaScript(ES6)
  - RWD
  - TailwindCSS
  - Vite
  - ESlint(airbnb)
  - prettier

- 模擬資料庫:
  - json-server-auth
  - Vercel -> Render
